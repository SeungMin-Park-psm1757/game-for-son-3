## ✨ 핵심 특징

### 📺 훈훈하고 재밌는 가족 스토리
- **4인의 가족 캐릭터**: 아빠, 엄마, 세연, 정우의 개성 넘치는 초상화와 대화 장면.
- **챕터별 진행**: 민물(Chapter 1) -> 연안(Chapter 2) -> 먼 바다(Chapter 3)로 이어지는 성장 이야기.
- **감동(?)의 엔딩**: 원양어선을 타려는 정우와 이를 말리는 가족들의 코믹한 결말.

### 🎣 실감 나는 낚시 시스템
- **스플라인 물리 낚싯줄**: 텐션에 따라 줄이 휘어지고 색상이 변하는 역동적인 비주얼.
- **피버 타임 (Fever Time)**: 3번 연속 실패 시 발생하는 무적의 피버 타임 시스템.
- **수학 퀴즈 에듀테인먼트**: 물고기를 잡을 때마다 나오는 간단한 퀴즈로 추가 골드 획득!
- **로또 행운**: 낚싯대 행운(Luck) 스탯에 따라 보너스 코인 주머니가 드랍됩니다.

### 🎨 고품질 픽셀 아트 및 비주얼 업그레이드
- **AI 기반 고해상도 픽셀 아트**: 아빠(군복), 엄마, 정우, 세연의 개성 넘치는 고품질 초상화와 스프라이트 적용.
- **역동적인 장비 진화**: 낚싯대 강화 시 캐릭터 색상이 변하는 대신, 낚싯대의 외형(목재->철->황금->터콰이즈->크리스탈->궁극)과 발광 효과가 화려하게 업그레이드됩니다.
- **비주얼 효과**: 낚싯대 강화 성공 시 화려한 파티클 효과와 루어(찌) 디자인 개선.

### 📱 독보적인 모바일 최적화 (Mobile Optimized)
- **가로/세로 최적화**: 어떤 방향에서도 플레이 가능하도록 버튼 크기와 레이아웃을 자동 조정합니다.
- **터치 스크롤 지원**: 모바일에서도 상점과 도감 리스트를 부드럽게 스크롤하여 모든 아이템을 확인 가능합니다.
- **최적화된 구도**: 모바일 내비게이션 바에 가려지지 않도록 초기화 버튼 위치를 조정하고, 캐릭터 시야를 확보했습니다.

### 👴 정겨운 한국 어부 NPC와 멘트
- **현지화된 NPC**: 이국적인 말투 대신 "오늘 물때가 좋네~" 처럼 정겨운 한국 낚시꾼 할아버지 대사로 리팩토링.
- **다양한 격려 멘트**: 물고기를 놓쳐도 "물고기가 메롱~ 하고 도망갔네? 😜" 등 10가지 이상의 랜덤 멘트로 아이의 재미를 유발합니다.

### ⚠️ 데이터 관리 및 품질
- **보안 초기화 버튼**: 첫 화면 하단에 안전하게 배치된 초기화 버튼.
- **비밀번호 보호**: 비밀번호 "5" 입력 시스템으로 아이의 실수 방지.
- **자동 품질 보정**: 저해상도 에셋을 지능적으로 업스케일링하여 대화면에서도 깨짐 없는 픽셀 아트를 제공합니다.

## 💻 로컬에서 실행하기

1. 저장소를 클론합니다:
   ```bash
   git clone https://github.com/SeungMin-Park-psm1757/game-for-son-3.git
   ```
2. 프로젝트 폴더로 이동합니다:
   ```bash
   cd game-for-son-3
   ```
3. 로컬 서버를 실행합니다 (예: Live Server, npx serve 등):
   ```bash
   npx serve .
   ```
4. `http://localhost:3000` (또는 지정된 포트)에서 확인하세요.
