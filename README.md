# 🎣 정우의 낚시 대모험 (Jeongwoo's Fishing Adventure)

[![GitHub Pages Deployment](https://img.shields.io/badge/Live-Play_Now-brightgreen?style=for-the-badge&logo=github)](https://seungmin-park-psm1757.github.io/game-for-son-3/)

**"아빠는 나라를 지키러 부대로, 오빠는 우리 가족을 위해 바다로!"**  
정우가 배고픈 동생 세연이를 위해 낚시로 돈을 벌어오는 훈훈하고 코믹한 가족 이야기 중심의 픽셀 아트 낚시 게임입니다!

## ✨ 핵심 특징

### 📺 훈훈하고 재밌는 가족 스토리
- **4인의 가족 캐릭터**: 아빠, 엄마, 세연, 정우의 개성 넘치는 초상화와 대화 장면.
- **챕터별 진행**: 민물(Chapter 1) -> 연안(Chapter 2) -> 먼 바다(Chapter 3)로 이어지는 성장 이야기.
- **감동(?)의 엔딩**: 원양어선을 타려는 정우와 이를 말리는 가족들의 코믹한 결말.

### 🎣 실감 나는 낚시 시스템
- **스플라인 물리 낚싯줄**: 텐션에 따라 줄이 휘어지고 색상이 변하는 역동적인 비주얼.
- **피버 타임 (Fever Time)**: 3번 연속 실패 시 발생하는 무적의 피버 타임 시스템.
- **수학 퀴즈 에듀테인먼트**: 물고기를 잡을 때마다 나오는 간단한 퀴즈로 추가 골드 획득!
- **로또 행운**: 낚싯대 행운(Luck) 스탯에 따라 보너스 코인 주머니가 드랍됩니다.

### 🎨 레트로 픽셀 아트
- 픽셀 아트 렌더링을 최적화하여 선명하고 귀여운 물고기 실루엣과 캐릭터를 감상할 수 있습니다.

### 📱 모바일 최적화 (Mobile Optimized)
- **가로/세로 최적화**: 어떤 방향에서도 플레이 가능하도록 버튼 크기와 레이아웃을 조정했습니다.
- **터치 친화적 UI**: 상점, 도감 등 팝업 UI의 크기를 모바일 화면에 맞춰 최적화했습니다.

### ⚠️ 데이터 관리
- **보안 초기화 버튼**: 첫 화면 좌측 하단에 초기화 버튼이 있습니다.
- **비밀번호 보호**: 비밀번호 **"5"**를 입력해야만 데이터가 초기화됩니다. (아이가 실수로 지우는 것을 방지합니다!)

- **Engine**: Phaser 3 (JavaScript Game Framework)

## 💻 로컬에서 실행하기

1. 저장소를 클론합니다:
   ```bash
   git clone https://github.com/SeungMin-Park-psm1757/game-for-son-3.git
   ```
2. 프로젝트 폴더로 이동합니다:
   ```bash
   cd game-for-son-3
   ```
3. 로컬 서버를 실행합니다 (예: Live Server, npx serve 등):
   ```bash
   npx serve .
   ```
4. `http://localhost:3000` (또는 지정된 포트)에서 확인하세요.
